ブックマークレット

Chrome 拡張機能が使えない環境でも、ブックマークから同じ操作ができます（閲覧中のページの URL をこの Webclip に送り、サーバが HTML を取得して解析します）。

  1. 下のコードをすべてコピーする。
  2. ブラウザのブックマークバーに新しいブックマークを追加し、URL 欄に貼り付ける。
  3. クリップしたいページを開いた状態で、そのブックマークをクリックする。

※ 閲覧中タブの URL と タイトル（document.title） がクエリで渡り、clip.html から API に送られます。タイトルは登録時に優先されます。ポートを変えた場合はこのページを開き直してコードを取り直してください。
